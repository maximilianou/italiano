### (Verbo irregulare)
```
Andare: 
          presente      passato prossimo
io        vado          sono andato/andata 
tu        vai           sei andato/andata 
lui/lei   va            &egrave; andato/andata
noi       andiamo       siamo andati/andate
voi       andate        siete andati/andate
loro      vanno         sono andati/andate 
 
```
